# Aplicación desarrollada en **Next.js** y Api Mock de prueba.

## Instrucciones para ejecutar la aplicación

1- **Instala las dependencias**
```bash
   npm install
```

2- **Verifica la configuración del backend:**
Esto con el objetivo del correcto funcionamiento de la aplicación puesto que existe conexion de Api.

3- **Inicia la aplicación en modo desarrollo**

```bash
   npm run dev
```

4- **Abre el navegador y dirígete a:**

```bash
http://localhost:3000/app-layout/dashboard
```
o bien, el puerto de elección.

5- **(Opcional) Para compilar y ejecutar en modo producción:**
```bash
npm run build
npm start
```
