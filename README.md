# Aplicación desarrollada en **Next.js** 

## Instrucciones para ejecutar la aplicación
Sigue estos pasos para clonar, configurar y poner en marcha la aplicación en tu entorno local.

1- **Clonar el Repositorio**: 
Primero, clona el código fuente del repositorio y navega al directorio del proyecto.

```bash
   git clone https://github.com/RogerG03/web-challenge-frontend.git
```
_o bien, descargas el archivo y lo descomprimes._

Luego navega al Directorio:

```bash
   cd web-challenge-frontend
   cd banca-digital-frontend
```

2- **Instala las dependencias**
```bash
   npm install
```

3- **Verifica la configuración del backend:**
Esto con el objetivo del correcto funcionamiento de la aplicación puesto que existe conexion de Api.

4- **Inicia la aplicación en modo desarrollo**

```bash
   npm run dev
```

5- **Abre el navegador y dirígete a:**

```bash
http://localhost:3000/app-layout/dashboard
```
o bien, el puerto de elección.

6- **(Opcional) Para compilar y ejecutar en modo producción:**
```bash
npm run build
npm start
```
