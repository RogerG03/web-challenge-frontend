## Convenciones de Nombre

- **Rutas y URLs (solo dentro de app):** `kebab-case`  
  Ej: `/app-layout/transferencias`

- **Carpetas:** `snake_case`  
  Ej: `components/`, `service/`

- **Componentes React:** `PascalCase`  
  Ej: `SidebarMenu.tsx`, `TransferForm.tsx`

- **Funciones:** `camelCase`  
  Ej: `handleSubmit()`

- **Archivos JSON, config y doc:** `kebab-case`  
  Ej: `api-config.json`

- **Clases CSS / Tailwind:** `kebab-case`  
  Ej: `.card-container`

- **Archivos especiales de Next.js:**  
  `layout.tsx`, `page.tsx`
